# 🎨 Ateliê Digital – Maria Jove

Bem-vindo ao repositório do **Ateliê Digital Maria Jove**, um projeto desenvolvido para apresentar o trabalho artístico da pintora de forma elegante, moderna e responsiva.

O objetivo deste site é oferecer um espaço onde visitantes possam conhecer a artista, explorar seu portfólio e entrar em contato para solicitar pinturas personalizadas, proporcionando uma experiência visual que reflita a sensibilidade e a identidade presentes em suas obras.

---

## ✨ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando:

- ⚡ **Astro** — Framework moderno para criação de sites rápidos e otimizados.
- 🎨 **Tailwind CSS** — Framework utilitário para estilização responsiva e elegante.
- 🖼️ **Astro Assets** — Otimização automática de imagens.

## 🚀 Como executar o projeto

### 1. Clone o repositório

```bash
git clone <url-do-repositorio>
```

### 2. Entre na pasta do projeto

```bash
cd nome-do-projeto
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o servidor de desenvolvimento

```bash
npm run dev
```

O projeto ficará disponível em:

```
http://localhost:4321
```

---

## 📱 Visualizar em outros dispositivos

Para acessar o site pelo celular na mesma rede Wi-Fi:

```bash
npm run dev -- --host
```

Depois, abra no navegador do celular o endereço IP informado pelo Astro, por exemplo:

```
http://192.168.0.10:4321
```

## 🎨 Sobre o Projeto

O **Ateliê Digital** foi pensado para transmitir a essência do trabalho artístico da pintora por meio de um design minimalista, elegante e acolhedor.

O site apresenta:

- 🖌️ Página inicial com destaque para a artista;
- 👩‍🎨 Seção "Sobre";
- 🖼️ Galeria/Portfólio de pinturas;
- 📩 Área de contato para encomendas;
- 📱 Layout totalmente responsivo para computadores, tablets e smartphones.

A identidade visual utiliza tons terrosos e neutros para valorizar as obras e proporcionar uma navegação agradável e intuitiva.
